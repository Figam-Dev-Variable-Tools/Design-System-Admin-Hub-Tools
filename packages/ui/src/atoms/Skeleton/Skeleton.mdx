# Skeleton

로딩 중 콘텐츠가 들어올 자리를 미리 표시하는 자리표시 블록입니다.

## 언제 사용하나요?

- 데이터를 불러오는 동안 빈 화면 대신 자리를 보여줄 때
- 표·카드 리스트가 로딩되는 자리를 채울 때
- 아바타·썸네일 등 이미지가 들어올 자리를 표시할 때

## Props

<Controls />

| Prop         | Description                                                                                    |
| ------------ | --------------------------------------------------------------------------------------------- |
| `shape`      | 자리표시 형태입니다. `line` = 텍스트 한 줄, `circle` = 아바타·아이콘 자리, `block` = 카드·썸네일 자리. |
| `isAnimated` | 맥동 애니메이션 여부입니다. `false` 면 정지한 회색 블록이 됩니다.                              |

`line` 과 `block` 은 컨테이너 폭을 채우고, `circle` 만 정사각으로 고정됩니다.

## Accessibility

- 스켈레톤 한 장은 항상 화면 낭독기에 노출되지 않습니다 (`aria-hidden`).
- 로딩 중이라는 사실은 스켈레톤을 담은 영역(표·카드 리스트)이 `aria-busy` 로 알립니다.
- 비대화형 장식이라 포커스 순서에 들어가지 않습니다.
- `prefers-reduced-motion` 에서는 `isAnimated` 값과 무관하게 애니메이션이 꺼집니다.

## Notes

- 표의 행·열 반복은 호출부가 자기 표 구조로 만들고, 각 칸을 Skeleton 한 장이 채웁니다.
- `isAnimated={false}` 는 정적 스냅샷(VRT·인쇄)용 옵트아웃이며 접근성 처리의 대체물이 아닙니다.
