# ToggleSwitch

즉시 반영되는 ON/OFF 토글 스위치입니다.

## 언제 사용하나요?

- 목록·폼에서 노출 여부를 바로 켜고 끌 때
- 팝업·배너·상품·리뷰 등의 표시 상태를 전환할 때
- 저장 버튼 없이 값이 즉시 반영되어야 할 때

## Props

<Controls />

| Prop       | Description                                                       |
| ---------- | ----------------------------------------------------------------- |
| `checked`  | ON 상태 여부입니다. `onChange` 로만 바뀌는 제어 값입니다.         |
| `label`    | 스크린 리더용 이름입니다. 표 안에서 무엇을 켜는지 알립니다.       |
| `disabled` | 잠그고 흐리게 표시합니다. 값이 바뀌지 않습니다.                   |
| `busy`     | 요청 진행 중 잠금입니다. `disabled` 처럼 막고 진행 중을 알립니다. |
| `onLabel`  | ON 상태 문구입니다. 기본값 `'ON'`.                                |
| `offLabel` | OFF 상태 문구입니다. 기본값 `'OFF'`.                              |

## Behavior

- `disabled` 또는 `busy` 이면 값이 바뀌지 않고 `onChange` 도 발화하지 않습니다.
- `onChange` 는 다음 상태(`!checked`)를 인자로 전달합니다.
- ON/OFF 문구로 상태를 색과 글자로 함께 전달합니다.

## Accessibility

- `role="switch"` 로 상태를 `aria-checked` 에 노출합니다.
- `Space` · `Enter` 로 토글하며 `Tab` 으로 포커스합니다.
- `busy` 일 때 `aria-busy` 로 진행 중임을 알립니다.
- 보이는 라벨이 없어도 `label` 이 이름을 제공합니다.
