# SelectField

옵션 중 하나를 고르는 드롭다운 컨트롤입니다. 네이티브 `<select>` 를 그대로 대체합니다.

## 언제 사용하나요?

- 정해진 목록에서 하나를 선택할 때
- 폼에서 구분·상태·카테고리 같은 값을 고를 때
- 목록을 좁히는 필터 드롭다운이 필요할 때

## Props

<Controls />

| Prop        | Description                                                             |
| ----------- | ---------------------------------------------------------------------- |
| `isInvalid` | 오류 상태로 표시합니다. 붉은 테두리를 내고 `aria-invalid` 를 붙입니다.  |
| `children`  | 표시할 `<option>` 들입니다. 호출부가 직접 넣습니다.                     |

표준 `<select>` 속성(`value`·`defaultValue`·`onChange`·`name`·`id`·`disabled`·`required`·`ref`·`aria-*` …)은 그대로 전달됩니다. 제어형 `value` 와 RHF `register` 를 모두 쓸 수 있습니다. `className`·`style` 은 받지 않습니다.

## Behavior

- 오류 메시지는 이 컨트롤이 아니라 감싸는 FormField 가 렌더합니다.
- `required` 를 주면 네이티브 `required` 와 `aria-required` 를 함께 냅니다.

## Accessibility

- 네이티브 `<select>` 이므로 `combobox` 역할과 키보드 조작(`Tab`·`↑`·`↓`·`Enter`·`Space`)이 기본 동작합니다.
- 키보드 포커스 시 포커스 링이 표시됩니다.
- `isInvalid` 는 색상뿐 아니라 `aria-invalid` 로도 오류를 알립니다.
- 커스텀 chevron 은 장식(`aria-hidden`)이며 클릭은 아래 `<select>` 로 통과합니다.
