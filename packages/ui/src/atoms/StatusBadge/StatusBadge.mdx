# StatusBadge

상태를 색(tone)과 라벨(label)로 함께 표시하는 비대화형 배지입니다.

## 언제 사용하나요?

- 목록의 상태 열에서 항목 상태를 표시할 때 (게시·예약·만료·임시저장 등)
- 좋음/주의/위험/정보 같은 의미를 색으로 구분해 보여줄 때
- 클릭·포커스가 필요 없는 단순 상태 표시가 필요할 때

## Props

<Controls />

| Prop    | Description                                                                                        |
| ------- | ------------------------------------------------------------------------------------------------- |
| `tone`  | 색 의도입니다. `neutral·success·warning·danger·info` 중 하나로 중립/좋음/주의/위험/정보를 표현합니다.    |
| `label` | 상태 문구입니다. 색만으로 의미를 전달하지 않도록 상태 의미를 담는 텍스트를 넣습니다. 예: '게시', '만료'. |

호출부가 도메인 상태(예: '게시')를 `tone`에 매핑합니다. 배지 자체는 무엇의 상태인지 알지 못합니다.

## Accessibility

- 색만으로 의미를 전달하지 않고 `label` 문구가 상태 의미를 함께 전달합니다 (WCAG 1.4.1).
- 비대화형이라 포커스 순서에 들어가지 않습니다.
- 어떤 항목의 상태인지는 인접한 제목·행이 문맥으로 제공합니다.
