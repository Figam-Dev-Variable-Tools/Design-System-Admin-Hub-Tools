# TriStateCheckbox

선택·미선택·부분선택 3상태를 가지는 체크박스입니다. 전체선택 헤더나 그룹 선택처럼 하위 항목이 일부만 선택된 상태를 나타낼 때 씁니다.

## 언제 사용하나요?

- 목록 표의 전체선택 헤더 (일부만 선택되면 부분선택으로 표시)
- 권한 매트릭스의 그룹(부모) 행·열 선택
- 하위 항목의 선택이 섞여 있음을 한눈에 보여줄 때

## Props

<Controls />

| Prop            | Description                                                              |
| --------------- | ----------------------------------------------------------------------- |
| `checked`       | 선택 여부입니다. 제어 값이라 `onChange` 로만 바뀝니다.                   |
| `indeterminate` | 부분선택입니다. `checked` 보다 우선해 표시됩니다.                        |
| `disabled`      | 비활성입니다. `onChange` 가 발화하지 않습니다.                          |
| `label`         | 보이는 라벨이 없을 때 접근 가능한 이름(`aria-label`)을 줍니다.          |
| `labelledBy`    | 라벨 요소의 id 입니다. 보이는 텍스트가 있으면 이걸 씁니다.               |
| `describedBy`   | 비활성 사유 등 부가 설명 요소의 id 입니다.                              |
| `onChange`      | 다음 체크 상태(`boolean`)를 인자로 호출됩니다.                          |

## Behavior

- `indeterminate` 가 `true` 면 `checked` 보다 우선해 부분선택으로 표시됩니다.
- `disabled` 면 클릭해도 `onChange` 가 발화하지 않고, `indeterminate` 표시도 함께 꺼집니다.

## Accessibility

- 부분선택은 `aria-checked="mixed"` 로 스크린리더에 알립니다.
- 키보드 `Space` 로 토글, `Tab` · `Shift+Tab` 으로 이동합니다.
- 보이는 라벨이 있으면 `labelledBy`, 없으면 `label` 로 이름을 줍니다.

## Notes

- 제어 컴포넌트입니다 — `checked` 를 직접 상태로 관리하고 `onChange` 로 갱신하세요.
