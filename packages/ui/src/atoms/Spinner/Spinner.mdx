# Spinner

작업이 진행 중임을 알리는 원형 로딩 표시입니다.

## 언제 사용하나요?

- 버튼이 작업을 처리하는 중일 때
- 영역·패널의 콘텐츠를 불러오는 중일 때
- 짧은 대기 상태를 시각적으로 알릴 때

## Props

<Controls />

| Prop    | Description                                                                                  |
| ------- | ------------------------------------------------------------------------------------------- |
| `size`  | 지름입니다. `inherit`(기본)은 부모 글자 크기를 따르고, `sm`/`md`/`lg`는 고정 크기입니다.     |
| `label` | 스크린리더에 낭독할 진행 문구입니다. 비우면 장식으로 처리됩니다.                             |

색은 `currentColor`를 따르므로 부모의 글자색과 같은 색으로 표시됩니다.

## Accessibility

- `label`이 있으면 `role="status"`로 표시되고 그 이름이 낭독됩니다.
- `label`이 비어 있으면(기본) 장식으로 보고 `aria-hidden="true"` 처리됩니다.
- `prefers-reduced-motion`을 지원합니다 — 회전이 멈춥니다.

## Notes

- 버튼 안에서는 버튼이 이미 `aria-busy`로 로딩을 알리므로 `label` 없이(장식으로) 사용합니다.
- 독립된 로딩 영역에서 쓸 때만 `label`을 주어 상태로 승격시킵니다.
