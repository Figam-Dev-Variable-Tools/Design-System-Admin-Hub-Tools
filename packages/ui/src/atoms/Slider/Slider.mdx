# Slider

값 표시가 옆에 붙은 단일 범위 입력 컴포넌트입니다.

## 언제 사용하나요?

- 숫자 값을 정해진 범위 안에서 조절할 때
- 여백·글자 크기·반경·크기 등 연속적인 값을 다룰 때
- 현재 값을 단위와 함께 바로 보여주고 싶을 때

## Props

<Controls />

| Prop       | Description                                                          |
| ---------- | ------------------------------------------------------------------- |
| `value`    | 현재 값입니다. 제어 값이며 `onChange` 로만 바뀝니다.                |
| `min`      | 허용 최솟값입니다.                                                  |
| `max`      | 허용 최댓값입니다.                                                  |
| `step`     | 증감 단위입니다. 화살표 키 한 번이 움직이는 크기이기도 합니다.      |
| `label`    | 스크린 리더용 이름입니다. 보이는 라벨이 없는 자리에서 사용합니다.   |
| `unit`     | 값 옆에 붙는 단위 표기('px' 등)입니다. 비우면 숫자만 보입니다.      |
| `disabled` | 비활성 상태로 잠그고 흐리게 표시합니다. `onChange` 가 발화하지 않습니다. |

## Behavior

<Story of={SliderStories.FiresWhenEnabled} />
<Story of={SliderStories.BlockedWhenDisabled} />

- 활성 상태에서 값을 옮기면 `onChange` 가 새 값(정수)으로 발화합니다.
- `disabled` 이면 값이 잠기고 `onChange` 가 발화하지 않습니다.

## Accessibility

- 키보드로 조작합니다: `Tab` · `←` · `→` · `↑` · `↓` · `Home` · `End`.
- 네이티브 `<input type="range">` 라 `role=slider` 와 `aria-valuemin/max/now` 를 브라우저가 제공합니다.
- 보이는 라벨이 없으므로 `label` 을 `aria-label` 로 전달합니다.
- 옆에 붙는 값 표시는 `aria-hidden` 장식이라 중복 통지가 없습니다.

## Notes

- 값 표시는 자릿수가 바뀌어도 트랙이 흔들리지 않도록 고정 폭입니다.
