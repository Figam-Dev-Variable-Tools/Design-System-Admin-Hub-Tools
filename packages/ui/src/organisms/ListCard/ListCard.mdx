# ListCard

제목·개수·목록 행을 담는 카드입니다.

## 언제 사용하나요?

- 대시보드에서 최근 주문·문의·상담 등 도메인 목록을 미리 보여줄 때
- 목록 옆에 전체 건수를 뱃지로 함께 보여줄 때
- 행을 클릭해 상세로 이동하는 요약 위젯이 필요할 때

## Props

<Controls />

| Prop         | Description                                                                        |
| ------------ | --------------------------------------------------------------------------------- |
| `title`      | 카드 제목입니다.                                                                  |
| `count`      | 제목 옆 뱃지에 표시할 건수입니다. `rows.length` 와 달라도 됩니다(전체 건수 vs 미리보기). |
| `rows`       | 표시할 행 목록입니다. `meta` 는 보조 정보, `href` 가 있으면 행 전체가 링크가 됩니다. |
| `loading`    | 로딩 중 스켈레톤 행을 표시하고 행 클릭을 막습니다.                                 |
| `empty`      | `rows` 가 비었을 때 보여줄 문구입니다.                                             |
| `icon`       | 각 행 앞에 붙는 아이콘입니다.                                                      |
| `onRowClick` | 행을 클릭할 때 `{ id, event }` 로 호출됩니다.                                      |

## Behavior

- 행에 `href` 가 있으면 실제 `<a>` 로 렌더되어 새 탭·가운데 클릭·URL 미리보기가 동작합니다.
- `onRowClick` 은 `{ id, event }` 를 넘기므로, 호출부에서 `event.preventDefault()` 후 `navigate()` 로 SPA 내비게이션을 가로챌 수 있습니다.
- `loading` 중에는 행이 렌더되지 않고 `onRowClick` 도 발화하지 않습니다.
- `rows` 가 빈 배열이면 `empty` 문구를 렌더합니다.

## Accessibility

- `Tab` · `Shift+Tab` 으로 각 행에 포커스가 이동하고 `:focus-visible` 링이 보입니다.
- 카드 제목이 목록 영역의 접근성 이름이 됩니다.
- `loading` 중에는 `aria-busy` 가 설정됩니다.

## Notes

- Card · Badge · ListRow 를 조립한 위젯입니다.
