# StatsCard

KPI·지표 카드 — 로딩·에러 상태를 포함한 통계 카드 껍데기입니다.

## 언제 사용하나요?

- 대시보드에서 지배적인 KPI 수치를 보여줄 때
- 헤더(제목 + 기간 토글 등 액션) 아래에 차트·표를 담을 때
- 데이터를 불러오는 동안 스켈레톤을 보여줄 때
- 조회 실패를 카드 안에서 알려야 할 때

## Props

<Controls />

| Prop       | Description                                                            |
| ---------- | --------------------------------------------------------------------- |
| `title`    | 카드 제목입니다. (필수)                                                |
| `action`   | 헤더 우측 액션 슬롯 — 기간 토글이나 버튼. 로딩·에러 중에도 유지됩니다. |
| `value`    | KPI 대표 수치. 포맷팅된 문자열을 넘기면 본문 위에 크게 표시됩니다.     |
| `children` | 본문 슬롯 — 차트·표 등. (필수)                                         |
| `loading`  | 본문을 스켈레톤으로 대체합니다.                                        |
| `error`    | 빈 문자열이 아니면 본문 대신 에러 문구를 보여줍니다. `loading` 보다 우선합니다. |

## Behavior

- `loading` 이면 본문만 스켈레톤으로 바뀌고, 액션 슬롯은 그대로 유지됩니다. 로딩 중 비활성이 필요하면 호출부가 슬롯 컴포넌트에 `disabled` 를 줍니다.
- `error` 가 있으면 본문 대신 에러 문구를 보여주며, `loading` 보다 우선합니다.
- `value` 는 호출부가 포맷을 끝낸 문자열을 넘깁니다 — 카드는 숫자를 다루지 않습니다.

## Accessibility

- 카드는 제목으로 접근성 이름을 얻습니다.
- 에러 문구는 스크린리더가 즉시 읽도록 안내됩니다.
- 로딩 중에는 진행 상태를 알립니다.
