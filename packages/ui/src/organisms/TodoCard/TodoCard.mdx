# TodoCard

할 일·처리 대기 항목을 한 카드에 모아 개수와 링크로 보여주는 업무 위젯입니다.

## 언제 사용하나요?

- 처리 대기 건수를 한눈에 보여줄 때
- 각 항목에서 해당 목록으로 바로 이동하고 싶을 때
- 대시보드에 '오늘의 할일' 요약을 놓을 때

## Props

<Controls />

| Prop        | Description                                                        |
| ----------- | ----------------------------------------------------------------- |
| `title`     | 카드 제목입니다. 기본값은 `'오늘의 할일'`.                         |
| `items`     | 표시할 항목 목록입니다. 호출부(Pages)가 직접 주입합니다.          |
| `loading`   | 로딩 중이면 스켈레톤을 보여주고 항목 클릭을 막습니다.             |
| `showTotal` | 제목 옆에 전체 count 합계를 Badge 로 표시할지 여부입니다.          |
| `onItemClick` | 항목을 클릭할 때 `{ key, event }` 로 호출됩니다.                |

각 항목은 `count > 0` 이면 강조색, `0` 이면 흐리게 렌더됩니다. `href` 가 있으면 링크, 없으면 버튼으로 표시됩니다.

## Behavior

- 항목 클릭 시 `onItemClick` 에 `{ key, event }` 를 넘깁니다. 호출부는 `event.preventDefault()` 후 `navigate()` 로 이동을 가로챌 수 있습니다.
- `loading` 상태에서는 항목이 렌더되지 않고 `onItemClick` 도 발화하지 않습니다.

## Accessibility

- 카드 제목이 영역의 접근성 이름이 됩니다.
- `Tab` · `Shift+Tab` 으로 항목을 이동하고 `Enter` 로 실행합니다.
- 로딩 중에는 `aria-busy` 로 안내합니다.
