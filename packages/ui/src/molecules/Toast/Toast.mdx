# Toast

작업 결과를 사용자에게 짧게 알려주는 피드백 컴포넌트입니다.

## 언제 사용하나요?

- 작업이 성공했을 때
- 오류가 발생했을 때
- 작업이 취소되었을 때
- 사용자에게 짧은 상태를 알려줄 때

## Props

<Controls />

| Prop         | Description                                                            |
| ------------ | --------------------------------------------------------------------- |
| `kind`       | 토스트의 종류입니다. 종류에 따라 아이콘·색상·자동 닫힘 여부가 달라집니다. |
| `message`    | 사용자에게 표시할 메시지입니다.                                        |
| `onRetry`    | 제공하면 오류 토스트에 '다시 시도' 버튼이 표시됩니다.                   |
| `onDismiss`  | 토스트를 닫을 때 호출됩니다.                                           |

## Behavior

| kind        | Auto dismiss |
| ----------- | ------------ |
| `success`   | 4s           |
| `info`      | 4s           |
| `cancelled` | 2s           |
| `error`     | 없음         |

`error` 는 사용자가 닫거나 다시 시도할 때까지 유지됩니다.

## Accessibility

- 오류는 우선순위가 높은 방식으로 안내됩니다.
- 그 외 토스트는 일반 상태 알림으로 안내됩니다.
- 닫기 버튼에는 접근 가능한 이름이 제공됩니다.
- `prefers-reduced-motion` 을 지원합니다.

## Notes

- 여러 개의 Toast 관리(큐·위치·최대 개수)는 ToastProvider 가 담당합니다.
