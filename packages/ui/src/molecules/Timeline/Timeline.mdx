# Timeline

시간순으로 쌓인 이벤트 이력을 목록으로 보여주는 컴포넌트입니다.

## 언제 사용하나요?

- 문의·티켓·주문 등 처리 이력을 시간순으로 보여줄 때
- 각 이벤트의 상태를 배지 톤·라벨로 구분해 알릴 때
- 작성자와 시각이 있는 활동 로그를 표시할 때

## Props

<Controls />

| Prop         | Description                                                                              |
| ------------ | --------------------------------------------------------------------------------------- |
| `events`     | 표시할 이벤트 목록입니다. 각 항목은 시각(`at`)·배지 톤/라벨·작성자·본문을 담습니다. 빈 배열이면 `emptyLabel` 이 표시됩니다. |
| `label`      | 목록의 이름입니다. 스크린 리더가 읽어 주는 접근 가능한 이름이 됩니다.                                  |
| `emptyLabel` | 이벤트가 없을 때 표시할 문구입니다.                                                          |

- 각 페이지는 자기 도메인 이벤트(문의·주문 등)를 위 형태로 매핑해 넘깁니다.
- `at` 은 ISO 시각 문자열이며 `YYYY-MM-DD HH:mm`(로컬 타임존)으로 표시됩니다.

## Accessibility

- 목록은 `<ol>` 로 렌더되어 순서 있는 목록으로 안내됩니다.
- 각 이벤트의 상태는 배지 색과 라벨 문구로 함께 전달됩니다(색에만 의존하지 않음).
- 정적 이력 목록이라 상호작용 키가 없으며 탭 순서에 들어가지 않습니다.
