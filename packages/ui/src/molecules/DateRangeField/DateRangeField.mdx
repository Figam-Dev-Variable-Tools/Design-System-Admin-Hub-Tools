# DateRangeField

시작일과 종료일을 '~' 로 이어 하나의 기간으로 입력받는 컴포넌트입니다.

## 언제 사용하나요?

- 노출·행사·프로모션 기간(시작~종료)을 입력받을 때
- 시작일과 종료일을 한 벌로 묶어 검증·오류를 함께 보여줄 때
- 필수/선택 기간 입력을 폼 안에서 일관되게 그릴 때

## Props

<Controls />

| Prop            | Description                                                          |
| --------------- | ------------------------------------------------------------------- |
| `label`         | 그룹 라벨. 각 칸의 숨김 라벨('… 시작일'·'… 종료일')도 여기서 파생됩니다. |
| `startValue`    | 시작일 값입니다 (`YYYY-MM-DD`).                                      |
| `endValue`      | 종료일 값입니다 (`YYYY-MM-DD`).                                      |
| `onStartChange` | 시작일이 바뀌면 새 문자열로 호출됩니다.                             |
| `onEndChange`   | 종료일이 바뀌면 새 문자열로 호출됩니다.                             |
| `required`      | 필수 표시(\*)를 붙이고 두 입력 모두 필수로 만듭니다.                |
| `disabled`      | 두 날짜 입력을 함께 비활성화합니다.                                 |
| `error`         | 인라인 오류 메시지. 있으면 오류로 표시되고 힌트는 가려집니다.       |
| `hint`          | 보조 안내. 오류가 없을 때만 표시됩니다.                             |

## Behavior

- 종료≥시작 같은 범위 규칙은 이 컴포넌트가 판정하지 않습니다 — 호출부에서 검증한 결과를 `error` 로 내려주면 그대로 표시합니다.
- `error` 가 있으면 힌트 대신 오류가 표시되고, 두 입력이 오류 스타일로 바뀝니다.

## Accessibility

- 두 칸은 하나의 그룹으로 묶이고, 각 입력에는 시작일·종료일을 구분하는 접근 가능한 이름이 있습니다.
- `Tab`·`Shift+Tab` 으로 시작일 → 종료일 순서로 이동합니다.
- 오류는 색뿐 아니라 메시지 텍스트로 함께 전달됩니다.
