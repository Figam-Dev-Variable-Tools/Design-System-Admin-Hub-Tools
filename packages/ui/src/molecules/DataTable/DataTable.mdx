# DataTable

요약(합계) 행·단위 표기·0 값 흐림을 갖춘 범용 데이터 표입니다.

## 언제 사용하나요?

- 일자·항목별 수치를 표로 나열할 때
- 합계·평균 같은 요약 행을 함께 보여줄 때
- 컬럼마다 단위(건·원·명)를 붙여야 할 때
- 0 값을 흐리게 처리해 유의미한 수치를 먼저 보이게 하고 싶을 때

## Props

<Controls />

| Prop          | Description                                                                                     |
| ------------- | ---------------------------------------------------------------------------------------------- |
| `columns`     | 컬럼 정의 배열. `align`(기본 `right`)·`unit`(요약 행 단위)·`unitInBody`(true면 본문 행에도 단위) |
| `rows`        | 본문 행 배열. 각 행은 `columns[].key` 를 키로 갖는다. 빈 배열이면 `empty` 문구를 표시한다.       |
| `rowKey`      | 행을 식별하는 컬럼 키. 이 컬럼은 `th[scope=row]` 로 렌더된다.                                   |
| `summaryRows` | `tfoot` 에 그리는 합계/요약 행. 빈 배열이면 요약 행을 그리지 않는다.                            |
| `caption`     | 표의 목적 설명. 시각적으로 숨기되 스크린리더에는 노출한다.                                       |
| `dimZero`     | 0 이하의 수치 셀을 흐리게 처리한다. (기본 `true`)                                               |
| `empty`       | `rows` 가 빈 배열일 때 표시할 문구.                                                             |

## Accessibility

- 표는 `caption` 을 접근성 이름으로 갖는다. (시각적으로만 숨김)
- 헤더 셀은 `scope=col`, `rowKey` 컬럼의 본문·요약 셀은 `scope=row` 로 연결된다.
