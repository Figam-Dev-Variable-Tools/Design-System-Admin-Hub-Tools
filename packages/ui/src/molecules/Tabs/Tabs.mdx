# Tabs

여러 패널 사이를 탭으로 전환하는 하단 밑줄형 탭 목록입니다.

## 언제 사용하나요?

- 한 화면에서 여러 영역을 탭으로 나눠 보여줄 때
- 목록의 상태·범위를 탭으로 걸러 볼 때
- 선택된 탭만 강조하고 나머지는 대기 상태로 둘 때

## Props

<Controls />

| Prop        | Description                                            |
| ----------- | ----------------------------------------------------- |
| `value`     | 선택된 탭의 id. `items[].id` 중 하나여야 합니다.      |
| `items`     | 탭 목록(`{ id, label }`).                             |
| `ariaLabel` | 탭 목록의 접근 가능한 이름 (예: '업무 영역').         |
| `onChange`  | 탭이 선택되면 해당 탭의 id 로 호출됩니다.             |

## Behavior

- 탭을 클릭하면 `onChange` 가 선택된 탭의 id 로 발화합니다.
- 선택된 탭만 액션 컬러 밑줄을 갖고, 비선택 탭도 같은 두께의 투명 밑줄을 둬 전환 시 라벨이 밀리지 않습니다.
- 탭 패널은 이 컴포넌트가 렌더하지 않습니다 — `aria-controls` 로 바깥 패널을 가리키므로 패널은 조립하는 쪽에서 렌더합니다.

## Accessibility

- `ArrowLeft` · `ArrowRight` 로 이전·다음 탭으로 이동하며 선택과 포커스가 함께 옮겨갑니다.
- `Home` · `End` 로 첫·마지막 탭으로 바로 이동합니다.
- 로빙 tabindex — 선택된 탭만 탭 순서에 있고 나머지는 건너뜁니다.
- 현재 탭은 `aria-selected` 로 안내됩니다.

## Notes

- 패널 id 규약은 `tabPanelId(id)` 로 공개됩니다. 그 id 로 패널이 실제 존재하는지는 조립하는 쪽의 책임입니다.
