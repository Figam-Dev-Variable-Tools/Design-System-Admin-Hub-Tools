# 비기능 명세서 템플릿

> **이 파일은 템플릿이다.** 복사해서 `docs/NFRS/<폴더>/<잎>.md` 로 저장하고 `〈…〉` 를 채운다.
> **인용부호(`>`)로 시작하는 안내 줄은 전부 지운다.**
>
> **절 이름과 순서를 바꾸지 않는다.** 여덟 절 전부를 갖는다.
> 각 절은 `공통 준수` + **이 자원만의 차이**를 적는다 — 공통과 같은 내용을 다시 쓰지 않는다([`README.md`](README.md) 규칙 N-1).
>
> 작성 원칙: 기획자 관점 · 표 형식 우선 · **API 는 호출 목적과 연계 여부만** ·
> **DB 설계·SQL·알고리즘 금지** · 재지 않은 값을 적지 않는다.
>
> 판정 기준의 정본: [`../reference/quality-bar.md`](../reference/quality-bar.md)
