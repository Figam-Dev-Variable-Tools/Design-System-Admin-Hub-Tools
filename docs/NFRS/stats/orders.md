# 주문 통계 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 주문 통계 |
| 화면 ID | `SCR-STATS-ORDERS` |
| 메뉴 경로 | 분석 · 운영 > 통계 > 주문 통계 |
| 작성자 | Admin Hub 문서화 |

| 버전 | 1.0 |
| 관련 기능 명세서 | [주문 통계](../../FSD/stats/orders/index.md) |

**이 자원의 성격** — 통계 6화면이 공유하는 껍데기 위의 읽기 전용 분석 화면이다. 성능·사용성·안정성·호환성·유지보수성의 값은 [방문자 통계](visitors.md) 와 **같다.** 이 문서는 **이 자원만의 차이**를 적는다. 차이의 뿌리는 둘이다: **주문 상태 어휘를 도메인과 공유하고, 결제를 쓰지 않으면 화면이 통째로 갈린다.**

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 전 항목 | [방문자 통계](visitors.md) 준수 — 백엔드 없음 · 쓰기 API 없음 · 조회 기간 상한 없음 · 미래 조회를 막지 않음 · 페이지 100행 · CSV 단일 형식 · 뒤로가기로 조건을 되돌릴 수 없음 |
| 결제 의존 | 결제를 쓰지 않으면 이 화면은 **주문을 보여 주지 않는다** — 문의·견적 지표로 대체된다 |
| 주문 상태 | 7종 고정이며 정본은 도메인 층이 갖는다 |
| 취소·교환·반품 | 상태가 아니라 각자의 건수다. 취소와 반품은 **배송중 시점**으로 갈린다 |
| 개별 주문 | 이 화면에서 열 수 없다 — 그건 주문 관리의 일이다 |
| 파일 용량 제한 · 업로드 확장자 | 해당 없음 — 파일 입력이 없다 |
