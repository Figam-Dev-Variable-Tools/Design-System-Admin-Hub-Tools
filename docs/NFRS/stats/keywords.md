# 검색어 분석 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 검색어 분석 |
| 화면 ID | `SCR-STATS-KEYWORDS` |
| 메뉴 경로 | 분석 · 운영 > 통계 > 검색어 분석 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [검색어 분석](../../FSD/stats/keywords/index.md) |

**이 자원의 성격** — 통계 6화면이 공유하는 껍데기 위의 읽기 전용 분석 화면이다. 성능·사용성·안정성·호환성·유지보수성의 값은 [방문자 통계](visitors.md) 와 **같다.** 이 문서는 **이 자원만의 차이**를 적는다. 차이의 뿌리는 셋이다: **플랜으로 잠기고, 검색 입력이 있는 유일한 통계 화면이며, 업계 관례에 없는 지표를 의도적으로 싣는다.**

## 3. 보안

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| 권한 | [방문자 통계](visitors.md) 준수 | 리소스 키는 `/stats/keywords` 이고 의미 있는 액션은 `read`·`export` 둘뿐이다 |
| **플랜(엔타이틀먼트)** | 공통 준수 | **이 자원은 잠긴다.** `stats.advanced`(표시명 `고급 통계`, 최소 티어 `프로`)가 이 경로를 지배한다 — `apps/admin/src/shared/entitlements/module-resources.ts:97-102` · `apps/admin/src/shared/entitlements/plan.ts:249-256` |
| 판정 순서 · 실패 방향 | 공통 준수 | 인증 → 플랜 → 권한. 플랜은 모르면 연다(fail-open) |
| ⚠ 개인정보 마스킹 | 공통 준수 | 마스킹 층이 **없다.** 검색어는 고객이 직접 친 문자열이라 이름·연락처·주문번호가 실려 올 수 있고, 그때 그 값이 표와 CSV 에 그대로 나간다. 화면 쪽에 걸러 내는 규칙이 코드에 없다 |
| 반출 데이터 | — | CSV 에 검색어 문자열이 그대로 담긴다(위 항목과 같은 위험) |
| 입력값 검증 | [방문자 통계](visitors.md) 준수 | 검색어에는 형식 제한이 없다 — 어떤 문자열이든 허용하고 앞뒤 공백만 무시한다 |

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 전 항목 | [방문자 통계](visitors.md) 준수 — 백엔드 없음 · 쓰기 API 없음 · 조회 기간 상한 없음 · 미래 조회를 막지 않음 · 페이지 100행 · CSV 단일 형식 · 뒤로가기로 조건을 되돌릴 수 없음 |
| 플랜 | `고급 통계` 모듈(최소 `프로` 티어)이 있어야 열린다 |
| 드릴다운 축 | 하나뿐이다(`검색어 순위`) |
| 검색어 유형 | `전체` · `결과 있음` · `결과 없음` 셋. 참조검색어 없음 행은 **`전체` 에서만** 보인다 |
| 검색어의 내용 | 고객이 직접 친 문자열이라 개인정보가 실려 올 수 있고, 마스킹 층이 없다(§3) |
| 결제 의존 | 결제를 쓰지 않아도 화면이 갈리지 않는다 — 구매 지표가 0 으로 그려진다 |
| 다음 행동 | 결과 없음 검색어에서 상품 등록·동의어 화면으로 가는 링크가 없다 |
| 파일 용량 제한 · 업로드 확장자 | 해당 없음 — 파일 입력이 없다 |
