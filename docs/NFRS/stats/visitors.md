# 방문자 통계 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 방문자 통계 |
| 화면 ID | `SCR-STATS-VISITORS` |
| 메뉴 경로 | 분석 · 운영 > 통계 > 방문자 통계 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [방문자 통계](../../FSD/stats/visitors/index.md) |

**이 자원의 성격** — **읽기 전용 분석 화면**이다. 저장·수정·삭제가 없어 그 축의 요구가 성립하지 않는 대신, ① 조회 조건의 재현성 ② 조회 상태 4분기의 정확성 ③ 숫자의 정직함(0 과 모름을 가르기) ④ 대량 반출 넷이 이 자원의 비기능 요구 전부다.

**통계 6화면이 같은 조회 조건 축과 같은 껍데기를 공유한다** — 성능·사용성·안정성·호환성·유지보수성의 값이 거의 전부 같다. **이 문서가 그 한 벌을 갖고**, [회원 통계](members.md) · [매출 통계](revenue.md) · [주문 통계](orders.md) · [유입 분석](traffic.md) · [검색어 분석](keywords.md) 은 자기 자원의 차이(플랜 잠금 · 결제 의존 · 검색 유무 · 고지 · 어휘)만 적는다.

아래 각 절은 [공통 기준](../_common.md) 대비 **이 자원만의 차이**를 적는다.
