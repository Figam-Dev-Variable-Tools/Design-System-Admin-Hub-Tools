# 회원 통계 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 회원 통계 |
| 화면 ID | `SCR-STATS-MEMBERS` |
| 메뉴 경로 | 분석 · 운영 > 통계 > 회원 통계 |
| 작성자 | Admin Hub 문서화 |

| 버전 | 1.0 |
| 관련 기능 명세서 | [회원 통계](../../FSD/stats/members/index.md) |

**이 자원의 성격** — 통계 6화면이 공유하는 껍데기 위의 읽기 전용 분석 화면이다. 성능·사용성·안정성·호환성·유지보수성의 값은 [방문자 통계](visitors.md) 와 **같다** — 그 문서가 한 벌을 갖는다. 이 문서는 **이 자원만의 차이**를 적는다. 가장 큰 차이는 **등급 어휘가 도메인과 어긋나 있다는 것**이다(§7).

## 1. 성능

| 응답 크기 | — | 응답이 **가입 경로 5종의 자리를 모두 채워** 온다 — 세그먼트 전환에 재조회가 필요 없는 대신 한 번에 받는 양이 경로 수만큼 늘어난다 — `apps/admin/src/pages/stats/members/types.ts:71-75` | 코드 확인 |
| 등급 구성의 축 | — | 등급별은 **네 줄짜리 스냅샷**이라 정렬·페이지네이션이 없다. 렌더 비용이 사실상 고정이다 | 코드 확인 |

## 3. 보안

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| 권한 | [방문자 통계](visitors.md) 준수 | 리소스 키는 `/stats/members` 이고 의미 있는 액션은 `read`·`export` 둘뿐이다 |
| 플랜(엔타이틀먼트) | 공통 준수 | 걸리지 않는다 — 심화 리포트 세 화면만 상위 플랜의 것이다 |
| 개인정보 마스킹 | 공통 준수 | **해당 없음** — 사람을 세기만 하고 개인을 식별하지 않는다. 계정·이름·연락처 열이 하나도 없다. 회원 목록 화면(`SCR-USERS-MEMBERS`)과 이 점이 결정적으로 다르다 |
| 반출 데이터 | — | CSV 에도 집계 수치만 담긴다 — 개인정보가 나가지 않는다 |
| 입력값 검증 | [방문자 통계](visitors.md) 준수 | 같다. 이 화면에는 검색 입력이 없다 |

## 7. 유지보수성

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| 전 항목 | [방문자 통계](visitors.md) 준수 | 같다 |
| ⚠ 등급 어휘가 두 벌이다 — 선재 결함 | 규칙의 정본은 하나여야 한다 | 이 화면은 회원 등급을 **네 종류**로 들고 있다(`일반`·`실버`·`골드`·`VIP`) — `apps/admin/src/pages/stats/members/types.ts:44-56`. 이 앱의 회원 등급 정본은 **세 종류**다(`일반회원`·`VIP`·`VVIP`) — `apps/admin/src/shared/domain/member.ts:20-26`. 같은 '회원 등급'이라는 말이 화면마다 다른 목록을 가리키며, 운영자가 이 화면에서 본 `골드 320명` 을 회원 목록에서 찾으면 그 등급 자체가 없다 |
| 같은 문제의 해결 선례 | — | **주문 통계는 이미 해결했다** — 주문 상태 어휘를 도메인 층으로 올리고 화면은 그것을 재수출한다. 어휘가 두 벌이면 같은 주문이 두 이름을 갖고 통계와 목록의 건수가 영원히 어긋난다는 것이 그 근거로 적혀 있다 — `apps/admin/src/pages/stats/orders/types.ts:20-26`. 회원 등급은 그 수렴이 되지 않았다 |
| 이 화면이 갖는 조립물 | — | 등급 목록 상수 하나뿐이다. 나머지는 껍데기 것이다 |

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 전 항목 | [방문자 통계](visitors.md) 준수 — 백엔드 없음 · 쓰기 API 없음 · 조회 기간 상한 없음 · 미래 조회를 막지 않음 · 페이지 100행 · CSV 단일 형식 · 뒤로가기로 조건을 되돌릴 수 없음 |
| 드릴다운 축 | `일자별` · `등급별` 둘뿐이다 |
| 가입 경로 | `이메일` · `네이버` · `카카오` · `구글` 넷 고정. 새 소셜 로그인이 붙으면 이 목록도 함께 고쳐야 한다 — `members/types.ts:24-30` |
| 회원 등급 | 네 종류 고정이며 **도메인과 어긋나 있다**(§7). 고객 설정 화면에서 바꿀 수 없다 |
| 등급 구성의 시점 | 기간 안의 변화가 아니라 **기간 말 스냅샷**이다 — `members/types.ts:65-69` |
| 개별 회원 | 이 화면에서 열 수 없다 — 사람을 세는 화면이지 찾는 화면이 아니다 |
| 파일 용량 제한 · 업로드 확장자 | 해당 없음 — 파일 입력이 없다 |
