# 매출 통계 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 매출 통계 |
| 화면 ID | `SCR-STATS-REVENUE` |
| 메뉴 경로 | 분석 · 운영 > 통계 > 매출 통계 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [매출 통계](../../FSD/stats/revenue/index.md) |

**이 자원의 성격** — 통계 6화면이 공유하는 껍데기 위의 읽기 전용 분석 화면이다. 성능·사용성·안정성·호환성·유지보수성의 값은 [방문자 통계](visitors.md) 와 **같다.** 이 문서는 **이 자원만의 차이**를 적는다. 차이의 뿌리는 셋이다: **돈을 다루고, 플랜으로 잠기며, 결제를 쓰지 않으면 화면이 통째로 갈린다.**

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 전 항목 | [방문자 통계](visitors.md) 준수 — 백엔드 없음 · 쓰기 API 없음 · 조회 기간 상한 없음 · 미래 조회를 막지 않음 · 페이지 100행 · CSV 단일 형식 · 뒤로가기로 조건을 되돌릴 수 없음 |
| 플랜 | `고급 통계` 모듈(최소 `프로` 티어)이 있어야 열린다 |
| 결제 의존 | 결제를 쓰지 않으면 이 화면은 **매출을 보여 주지 않는다** — 문의·견적 지표로 대체된다 |
| 결제수단 | `신용카드` · `계좌이체` · `가상계좌` · `간편결제` 넷 고정 |
| 과세 구분 | 과세 · 면세 · 영세 셋이며 **일자별 축에서만** 편다(수단별로는 갈리지 않는다) |
| 세무 용도 | **제출용 자료가 아니다** — 결제 시점의 집계라 세무 신고가 요구하는 귀속·공급가액 기준과 다르다. 그 사실은 화면에만 있고 파일에는 없다 |
| 대체 패널의 값 | 문의·견적 조회 이음매가 배선되기 전에는 **0 이 아니라 '모름'** 이며, 힌트 뒤에 그 사실이 붙는다 — `_shared/InquiryStatsPanel.tsx:56` |
| 파일 용량 제한 · 업로드 확장자 | 해당 없음 — 파일 입력이 없다 |
