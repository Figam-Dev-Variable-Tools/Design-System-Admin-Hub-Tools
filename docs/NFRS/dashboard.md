# 대시보드 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 대시보드 |
| 화면 ID | `SCR-DASHBOARD` · `TAB-DASHBOARD-PRODUCTS` · `TAB-DASHBOARD-INQUIRIES` · `TAB-DASHBOARD-SALES` |
| 메뉴 경로 | 일반 관리 > 대시보드 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [대시보드](../FSD/dashboard/index.md) · [상품 탭](../FSD/dashboard/tab-products.md) · [문의 탭](../FSD/dashboard/tab-inquiries.md) · [영업 탭](../FSD/dashboard/tab-sales.md) |

**이 자원의 성격** — 로그인 후 랜딩이자 **읽기 전용** 화면이다. 쓰기 동작이 하나도 없어 저장·충돌·중복 제출 같은 축이 성립하지 않는 대신, **조회가 여럿 동시에 도는 화면**이라 로딩·실패·권한 분기가 이 자원의 전부다. 아래 각 절은 [공통 기준](_common.md) 대비 **이 자원만의 차이**를 적는다.

## 1. 성능

| 항목 | 요구 | 이 자원의 값 | 측정 방법 |
