# 공지사항 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 공지사항 |
| 화면 ID | `SCR-CONTENT-NOTICES` · `SCR-CONTENT-NOTICES-DETAIL` · `SCR-CONTENT-NOTICES-FORM` |
| 메뉴 경로 | 콘텐츠 관리 > 공지사항 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [목록](../../FSD/content/notices/index.md) · [상세](../../FSD/content/notices/detail.md) · [등록·수정](../../FSD/content/notices/form.md) |

**이 자원의 성격** — **콘텐츠 관리 묶음의 정본이다.** 이 묶음의 일곱 자원은 같은 데이터 소스 규약과 같은 한계를 공유하므로, 나머지 여섯 문서는 이 문서와의 **차이만** 적는다. 이 묶음 전체를 관통하는 사실이 셋 있다 — ① 쓰기가 저장되지 않는다 ② 쓰기 권한 게이팅이 없다 ③ 폼이 공용 컨트롤러를 쓰지 않아 충돌·422·중복 제출 방어가 없다. **뉴스만 그 셋에서 벗어난다**([뉴스 문서](news.md) 참조).
