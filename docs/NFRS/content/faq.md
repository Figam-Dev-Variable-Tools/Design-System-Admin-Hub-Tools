**이 자원의 성격** — [공지사항](notices.md)과 **같은 골격·같은 한계**를 갖는다(쓰기 미반영 · 권한 게이팅 없음 · 공용 폼 컨트롤러 미사용). 아래 각 절은 그 문서 대비 **다른 것만** 적는다. 다른 것은 넷이다: **분류를 운영자가 만든다** · **목록에서 순서를 바꾼다** · **목록에서 노출을 켜고 끈다** · **일부 쓰기는 실제로 반영된다**.

## 2. 사용성

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| UI 일관성 | 공통 준수 | 공지사항과 같다 — 표·필터 배치·폼 골격을 화면이 직접 조립한다 |
| 조회 상태 보존 | 공통 준수 | 카테고리·노출 여부·검색어·페이지를 주소가 소유한다 — `apps/admin/src/pages/content/faq/FaqPage.tsx:108-118` |
| 여러 건을 한 번에 정리하기 | 반복 작업을 왕복시키지 않는다 | 노출을 **표에서 바로** 켜고 끄고, 선택한 여러 건을 **확인 없이** 일괄 노출/숨김한다 — 되돌릴 수 있는 조작이라 확인을 세우지 않는다 |
| 순서를 손으로 바꾼다 | 드래그를 쓸 수 없는 사용자도 같은 일을 할 수 있다 | 드래그 손잡이와 **행마다의 위/아래 이동 버튼**을 함께 준다 — `apps/admin/src/pages/content/faq/components/FaqTable.tsx:193-201` |
| ⚠ 재정렬이 막힌 이유를 말하지 않는다 | 왜 못 하는지 알 수 있어야 한다 | 카테고리·노출 필터·검색어 중 하나라도 걸리면 **손잡이와 이동 버튼이 조용히 사라진다** — `apps/admin/src/pages/content/faq/FaqPage.tsx:154`. [인증서/특허](../company/certificates.md)는 같은 자리에서 사유를 문장으로 남긴다 |
| 표 설명이 실제 조작을 따라간다 | — | caption 이 재정렬 가능 여부에 따라 한 문장을 더하거나 뺀다 — `apps/admin/src/pages/content/faq/components/FaqTable.tsx:119-122` |
| 사용자 피드백 | 공통 준수 | **이 자원의 토스트가 10건으로 콘텐츠 관리에서 가장 많다**(성공 7 · 실패 3). 실패 셋에는 전부 '다시 시도'가 붙어 있고 실패 토스트는 자동으로 사라지지 않는다 |
| 삭제를 막는 이유를 숫자로 말한다 | — | 카테고리 팝업이 각 분류의 사용 건수를 배지로 보여 주고, 사용 중이면 휴지통을 잠그며 툴팁으로 이유를 말한다 — `apps/admin/src/pages/content/faq/components/ManageFaqCategoriesModal.tsx:102-104,276-283` |
| ⚠ 팝업의 조회 실패를 알리지 않는다 | `null`(모름)과 `[]`(없음)을 섞지 않는다 | 카테고리 목록 조회가 실패하면 **0건과 같은 화면**이 된다 — `등록된 카테고리가 없어요.` — `apps/admin/src/pages/content/faq/components/ManageFaqCategoriesModal.tsx:261` |
| ⚠ 선택지가 없을 때의 안내 | — | 카테고리가 0건이면 폼의 분류 select 에 고를 것이 없는데 **왜 없는지 말하지 않는다** — 제출해야 `카테고리를 선택하세요.` 가 뜬다 |
