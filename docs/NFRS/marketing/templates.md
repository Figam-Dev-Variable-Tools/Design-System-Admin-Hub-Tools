# 발송 템플릿 관리 — 비기능 명세서

| 항목 | 내용 |
|---|---|

**이 자원의 성격** — 마케팅 묶음에서 **가장 무거운 자원**이다. 화면이 다섯이고, 한 모델 위에 **네 종류의 편집기**가 얹혀 있으며, 그중 이메일은 **블록 빌더**라는 독립된 편집 환경을 갖는다. 발송 3종([SMS 발송](sms.md) 외)이 이 자원이 만든 것을 불러 쓴다.

## 1. 성능

| 항목 | 요구 | 이 자원의 값 | 측정 방법 |
|---|---|---|---|
| 응답 시간 | 공통 준수 | **측정하지 않음** — 백엔드가 없다 | 백엔드 연동 후 재작성 |

| 동시 사용자 | 공통 준수 | **측정하지 않음** | — |
| 대용량 데이터 처리 | 가상화 없이 대량 행을 그리지 않는다 | ⚠ 목록에 **페이지네이션도 가상화도 없다** | 코드 확인 |
| 편집 중 비용 | 타이핑이 화면을 붙잡지 않는다 | 이메일 빌더는 **되돌리기 이력을 메모리에 쌓는다** — 값이 바뀌는 모든 경로가 한 단위씩 쌓이며 **상한이 코드에 보이지 않는다.** 긴 편집 세션에서 메모리가 단조 증가한다 — `apps/admin/src/pages/marketing/message-templates/email/EmailBuilder.tsx:10-11` | 코드 확인 |
| 검증 비용 | — | 저장 시 **본문 전체를 훑어 치환변수 토큰을 뽑는다.** 본문 상한이 작아(문자 수천 바이트 · 카카오 1,000자대) 체감 비용은 낮다 | 코드 확인 |
| 미리보기 | — | 카카오·문자 미리보기는 같은 화면 안 컴포넌트라 추가 조회가 없다 | 코드 확인 |

| 접근성 | 공통 준수 | 큰 제목 칸은 **보이는 라벨 없이** 접근성 이름을 갖는다(제목이 둘이 되지 않게) — `TemplateEditorShell.tsx:133-134`. 잘못됨만 알리고 이유를 말하지 않는 입력을 만들지 않는다 — `TemplateEditorShell.tsx:145`. 되돌리기·다시하기에는 **눌림 상태를 주지 않는다**(일반 액션이라 토글로 읽히면 거짓이다) — `apps/admin/src/pages/marketing/message-templates/email/EmailToolbar.tsx:175-176` |

| 개인정보 마스킹 | — | **해당 없음** — 템플릿은 개인 데이터를 담지 않는다. 개인 정보가 들어가는 자리는 **치환변수**이며, 치환은 발송 시점에 일어난다(이 화면은 토큰만 갖는다) |
| 입력값 검증 | 화면 검증과 서버 검증을 구분한다 | 화면 검증만 있다. **가장 중요한 규칙이 여기 있다** — 카탈로그에 없는 치환변수는 **저장을 막는다.** 경고는 넘길 수 있고 넘길 수 있는 경고는 결국 넘기며, 통과시키면 수신자 화면에 `#{member.nmae}` 가 그대로 찍힌 뒤에야 알게 된다 — `apps/admin/src/pages/marketing/message-templates/validation.ts:98-112` |

## 5. 호환성

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| 지원 브라우저 | 공통 준수 | **확인하지 못했다** — 선언이 없다 |
| 모바일 지원 | 공통 준수 | 관리자 화면 전제. **이메일 빌더는 3단 구성이라 좁은 화면에서 가장 불리한 화면**이다 |
| 해상도 | 넓은 표·넓은 편집기 | 빌더가 [프리셋 레일][캔버스][속성 패널] 3단이며 **양쪽 사이드를 접을 수 있다** — 그것이 좁은 폭에 대한 유일한 대응이다 |
| 문자 인코딩 | 등급 판정이 브라우저를 타지 않는다 | 문자는 **EUC-KR 바이트**, 카카오는 **글자 수**로 센다. 두 축이 섞이지 않도록 카카오 규칙 파일은 바이트 계산기를 부르지 않고 문자 상한 상수도 재사용하지 않는다 — `apps/admin/src/pages/marketing/message-templates/kakao.ts:28-35` |
| 미리보기의 충실도 | — | ⚠ **실제 수신 화면과 같다고 보장하지 않는다** — 메일 클라이언트·카카오 앱의 렌더 차이를 검사하지 않는다 |

|---|---|
| 외부 시스템 연계 | 카카오 비즈니스(알림톡 심사) · 문자 발송사 · 메일 발송 서비스. **전부 연동되어 있지 않다** — 심사 상태는 화면이 들고 있는 값이며 카카오에서 받아 오지 않는다 |
