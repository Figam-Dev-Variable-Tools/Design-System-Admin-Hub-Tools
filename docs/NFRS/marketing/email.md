# 이메일 발송 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 이메일 발송 |
| 화면 ID | `SCR-MARKETING-EMAIL` · `SCR-MARKETING-EMAIL-FORM` |
| 메뉴 경로 | 비즈니스 > 마케팅 관리 > 이메일 발송 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [목록](../../FSD/marketing/email/index.md) · [등록·수정](../../FSD/marketing/email/form.md) |

**이 자원의 성격** — [SMS 발송](sms.md) 과 **같은 발송 3종**이다. 여덟 절의 값 대부분이 같다 — 그 문서가 한 벌을 갖는다. 이 문서는 **차이만** 적는다. 가장 큰 차이는 **플랜에 걸리는 방식**(§3)과 **성과 지표가 있다는 것**(§2)이다.

## 1. 성능

| 항목 | 요구 | 이 자원의 값 | 측정 방법 |
|---|---|---|---|
| 전 항목 | [SMS 발송](sms.md) 준수 | 같다 — **페이지네이션도 가상화도 없다** | 코드 확인 |
| 본문 길이 계산 | — | **바이트 등급 판정이 없다** — 이메일은 길이로 등급이 갈리지 않는다. SMS 폼이 입력마다 돌리는 계산이 여기에는 없다 | 코드 확인 |
| 성과 지표 | — | 열람률·클릭률은 **받은 값으로 계산한 비율**이라 추가 조회가 없다 — `apps/admin/src/pages/marketing/_shared/messaging.ts` 의 발송 성과 |

## 2. 사용성

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| 전 항목 | [SMS 발송](sms.md) 준수 | 같다 — 세그먼트 선택기 · 변수 삽입 바를 공유한다 |
| 성과가 목록에 보인다 | 보낸 뒤에 할 일이 있다 | 발송 이력에 **열람·클릭 비율**이 함께 뜬다. SMS 에는 그 축이 없다(문자에는 열람 개념이 없다) |
| 제목이 필수다 | — | 이메일은 제목이 수신함에서 유일한 단서라 필수다. SMS 는 등급에 따라 제목이 아예 없다 |
| 본문 편집 | — | 이 화면의 본문은 **평문·단순 입력**이다. 블록 빌더는 [발송 템플릿](templates.md) 자원의 것이며, 여기서는 만들어 둔 템플릿을 불러 쓴다 |

## 3. 보안

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| 권한 | [SMS 발송](sms.md) 준수 | **리소스 키만 다르다** — 잎 `/marketing/email` |
| 플랜(엔타이틀먼트) | 공통 준수 | ⚠ **SMS 와 다른 축이다** — `marketing.email` 의 최소 티어가 `basic` 이라, 하위 플랜에서는 **사라지지 않고 잠금 화면으로 뜬다** — `apps/admin/src/shared/entitlements/plan.ts:226-232`. 잠긴 메뉴에는 잠금 표시가 붙는다. **한 모듈 키가 두 화면을 덮는다** — 이 화면과 [뉴스레터](newsletters.md) 가 같은 스위치로 함께 열리고 함께 잠긴다 — `apps/admin/src/shared/entitlements/module-resources.ts:80-83` |
| 개인정보 마스킹 | — | **해당 없음** — 수신자를 세그먼트와 인원수로만 다룬다. 수신 이메일 주소 열이 화면에 없다 |
| 입력값 검증 | [SMS 발송](sms.md) 준수 | 같다. **광고 표기·야간 발송 제약은 이 자원에 걸리지 않는다**(문자·카카오의 규칙이다) |

## 4. 안정성

| 항목 | 요구 | 이 자원의 값 |

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 전 항목 | [SMS 발송](sms.md) 준수 — 백엔드 없음 · 페이지 크기 상한 없음 · 수신 거부 규칙 없음 · 나간 것은 되돌릴 수 없음 |
| 외부 시스템 연계 | 메일 발송 서비스. **연동되어 있지 않다** |
| 발송 시간 | **제한이 없다** — 광고성 문자의 야간 금지는 이 자원에 적용되지 않는다 |
| 첨부 파일 | 이 화면에 첨부 입력이 없다 |
| 성과 지표의 출처 | 열람·클릭은 **서버가 줄 값**이다. 지금 화면이 보여 주는 것은 계산된 비율뿐이며 집계 주체는 없다 |
