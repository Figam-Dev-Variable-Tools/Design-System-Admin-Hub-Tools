**이 자원의 성격** — **회사당 1건짜리 단일 문서**를 편집하는 폼이다. 목록도 페이지네이션도 없어 대량 데이터 축이 성립하지 않는 대신, ① **개정 토큰으로 동시 편집을 막고** ② **이미지 업로드가 있으며** ③ 저장된 값이 발송·로그인 화면으로 건너간다는 셋이 이 자원의 비기능 특성이다. 아래 각 절은 [공통 기준](../_common.md) 대비 **이 자원만의 차이**를 적는다.

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 외부 시스템 연계 | 고객이 보는 공개 사이트는 **이 리포 밖**이다. 복사 방지·모바일 확대 허용은 그쪽 규칙이라 이 어드민에서 결과를 확인할 수 없다 |
| API 제약 | **백엔드가 없다.** 조회·저장·업로드 셋 다 픽스처다. 계약은 `data-source.ts:77-82,92` 의 `TODO(backend)` 가 갖는다 |

| 파일 용량 제한 | 파비콘 **100KB** · 대표 이미지·비공개용 이미지 **5MB** — `validation.ts:36,39` |
