# 로그인 이력 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 로그인 이력 |
| 화면 ID | `SCR-USERS-LOGIN-HISTORY` |
| 메뉴 경로 | 일반 관리 > 사용자 관리 > 로그인 이력 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [로그인 이력](../../FSD/users/login-history/index.md) |

**이 자원의 성격** — **읽기 전용 감사 로그**다. 저장·수정·삭제가 하나도 없어 그 축의 요구가 성립하지 않는 대신, ① **불변성** ② **계정·이름·IP 의 반출** ③ **조회 구간의 정확성**(타임존) ④ **연속 실패 신호의 보존** 넷이 이 자원의 비기능 요구 전부다. 로그 섹션(`/logs/**`)과 성격이 같지만 **껍데기를 공유하지 않는다** — 이 화면은 자기 필터·자기 표를 갖는다.

아래 각 절은 [공통 기준](../_common.md) 대비 **이 자원만의 차이**를 적는다.
