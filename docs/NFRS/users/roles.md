# 권한 관리 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 권한 관리 |
| 화면 ID | `SCR-USERS-ROLES` · `POP-USERS-ROLES-ROLE-FORM` |
| 메뉴 경로 | 일반 관리 > 사용자 관리 > 권한 관리 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [권한 관리](../../FSD/users/roles/index.md) · [역할 추가·수정](../../FSD/users/roles/pop-role-form.md) |

**이 자원의 성격** — **이 앱의 권한 모델 자체를 정의하는 화면**이다. 다른 15묶음의 §9 가 여기서 정한 것을 참조한다. 비기능 관점에서 특별한 점은 셋이다: ① **네트워크를 타지 않는다**(상태가 브라우저 저장소에 있다) ② **저장 버튼이 없다** — 체크가 곧 저장이고 그 결과가 앱 전체의 메뉴·버튼 가시성으로 즉시 번진다 ③ **잠금(lock-out) 위험이 가장 큰 자원**이다 — 잘못 끄면 스스로를 잠글 수 있다.

아래 각 절은 [공통 기준](../_common.md) 대비 **이 자원만의 차이**를 적는다.

## 3. 보안

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| 인증 | 공통 준수 | 셸 안쪽이라 세션이 없으면 렌더되지 않는다 |
| 권한 | 공통 준수 | ⚠ **이 화면은 `read` 게이팅만 받는다.** 역할 추가·개명·삭제·매트릭스 편집·위젯 토글 어디에도 쓰기 권한 판정이 없다(`PermissionsPage.tsx` 가 `useRouteWritePermissions` 를 부르지 않는다). **읽기만 가진 역할이 권한 모델 전체를 고칠 수 있다** — 같은 섹션의 회원 관리·고객 설정·관리자 관리가 전부 쓰기 게이팅을 갖는 것과 대비된다 |

| 고아 참조 방지 | — | 배정된 운영자가 있는 역할은 삭제가 거절된다. **인원을 셀 수 없을 때도 거절한다**(fail-closed) — `roles.ts:331-338` |
| 저장 위치 | 공통 준수 | 브라우저 저장소에 **평문 JSON** 이다. 위조하면 이 화면의 판정이 통째로 우회된다 — 프런트 게이팅은 보안 경계가 아니라 UX 이며 실제 차단은 서버의 몫인데 **지금 서버가 없다** — `apps/admin/src/shared/permissions/RequirePermission.tsx:8-11` |

## 7. 유지보수성

| 항목 | 요구 | 이 자원의 값 |

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 외부 시스템 연계 | 없다 |
| API 제약 | **네트워크 요청이 0건이다.** 상태가 브라우저 저장소에 있고, 백엔드 자리는 주석으로만 표시되어 있다: `GET /api/roles` · `PUT /api/roles/:id/permissions` — `roles.ts:11` · `apps/admin/src/shared/permissions/PermissionProvider.tsx:23` |
| 데이터 접근 범위(scope) | 역할당 `전체`·`소속 부서`·`본인` 셋을 **저장만 한다.** 목록·상세 질의의 실제 필터링은 서버 몫으로 표시되어 있고, 지금은 이 값이 어떤 화면 동작도 바꾸지 않는다 — `apps/admin/src/pages/permissions/components/RoleHeaderCard.tsx:151-152` · `roles.ts:30-32` |
| 승인·반려 액션 | **권한 모델에 없다.** 액션은 다섯뿐이며, 어느 화면의 명세도 그 축을 쓸 수 없다 |
| 위젯 권한의 범위 | 대시보드 위젯 7개는 액션 개념이 없어 매트릭스가 아니라 단일 토글이다. **대시보드 밖의 화면에는 위젯 축이 없다** |
| 저장 범위 | 설정이 **이 브라우저에만** 저장된다. 다른 기기·다른 브라우저에서는 기본 역할로 시작하며, 화면이 그 사실을 안내문으로 밝힌다 |
| 감사 기록 | 권한 변경 이력이 남지 않는다(§4) |
| 파일 용량 제한 | 해당 없음 — 파일 입력이 없다 |
| 업로드 가능 확장자 | 해당 없음 — 같은 이유 |
