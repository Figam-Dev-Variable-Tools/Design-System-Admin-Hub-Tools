# API 로그 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | API 로그 |
| 화면 ID | `SCR-LOGS-API` |
| 메뉴 경로 | 분석 · 운영 > 로그 관리 > API 로그 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [API 로그](../../FSD/logs/api/index.md) · 공유 팝업 [로그 원문 보기](../../FSD/_common/pop-log-payload.md) |

**이 자원의 성격** — 로그 4화면이 공유하는 껍데기 위의 화면이다. 성능·사용성·안정성·호환성·유지보수성의 값은 [관리자 로그](admin.md) 와 **같다** — 그 문서가 껍데기 한 벌을 갖는다. 이 문서는 **이 자원만의 차이**를 적는다. 차이의 뿌리는 둘이다: **행위자가 사람이 아니라 키이고, 부피가 압도적으로 크다.**

## 1. 성능

| 항목 | 요구 | 이 자원의 값 | 측정 방법 |
|---|---|---|---|
| 전 항목 | [관리자 로그](admin.md) 준수 | 같다 — 같은 껍데기·같은 조회 엔진·같은 페이지 상한(100행, 기본 20) | 코드 확인 |
| ⚠ 데이터 부피 | — | **4화면 중 압도적으로 크다** — 관리자 로그의 수천 배로 기술되어 있다 — `apps/admin/src/pages/logs/api/types.ts:99-103`. 그런데 조회 상한(90일)·페이지 상한(100행)은 다른 화면과 **같은 값**이며, 이 자원만의 추가 제약이 코드에 없다 | 코드 확인 |
| 내보내기 소요 | — | 취소 경로의 존재 이유에 가장 가까운 화면이다 — 90일치는 길다 — `apps/admin/src/pages/logs/components/LogToolbar.tsx:104-105`. **소요 시간은 측정하지 않음**(백엔드가 없다) | 백엔드 연동 후 |
| 느린 호출의 기준 | — | 응답 시간 **1,000ms** 이상을 느린 것으로 본다 — `api/types.ts:95`. 1초는 사람이 '멈췄다'고 느끼기 시작하는 지점이고, API 에서는 이미 사고다 — `api/types.ts:91-94`. 이것은 이 앱의 성능 목표가 아니라 **표시 기준**이다 |

## 2. 사용성

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| 전 항목 | [관리자 로그](admin.md) 준수 | 같다 |
| 숫자 컬럼 표기 | — | **4화면 중 처음으로 숫자 컬럼이 등장한다.** `응답시간(ms)` 은 우측 정렬 + 등폭 숫자다 — `apps/admin/src/shared/ui/styles.ts:392-396`. 등폭이 아니면 1400 과 980 의 자릿수가 어긋나 어느 쪽이 큰지 눈으로 못 읽는다 — `apps/admin/src/pages/logs/api/ApiLogPage.tsx:7-10` |
| 단위의 자리 | — | 단위(`ms`)는 값이 아니라 **헤더에 올린다** — 값마다 단위가 따라다니면 자릿수 정렬이 다시 깨진다 — `ApiLogPage.tsx:9-10` |
| 상태를 색으로만 전달하지 않기 | 공통 준수 | 4xx·5xx 는 코드 옆에 **계열 이름을 글자로** 함께 적는다 — `ApiLogPage.tsx:26-38`. 느린 호출은 숫자 옆에 `느림` 이 붙는다 — `ApiLogPage.tsx:41-54` |
| 톤을 섞지 않기 | — | 5xx 는 danger, 4xx 는 warning 이다 — **5xx 는 우리 잘못이고 4xx 는 부르는 쪽의 잘못이라 같은 색으로 섞지 않는다** — `ApiLogPage.tsx:79`. 2xx 라도 느리면 warning 이 된다 |
| 요약 경고의 우선순위 | — | 5xx 를 먼저, 없으면 4xx 를 말한다. **느린 호출은 요약 줄에 올라오지 않는다** — 표의 표기와 정렬로만 찾는다 — `ApiLogPage.tsx:111-117` |
| 축의 입도 | — | 상태를 낱개 코드가 아니라 **계열(2xx/4xx/5xx)** 로 나눈다 — 운영자가 묻는 것은 '404 가 몇 건인가'가 아니다 — `api/types.ts:22-26` |
| 나가는 길 | — | ⚠ **이 화면에는 행에서 다른 화면으로 가는 링크가 없다.** 다른 두 화면이 갖는 '행에서 대상 화면으로'가 여기에는 성립하지 않고, 키를 다루는 화면(`SCR-SETTINGS-API-KEYS`)으로 가는 링크도 배선되어 있지 않다 |

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 전 항목 | [관리자 로그](admin.md) 준수 — 백엔드 없음 · 쓰기 API 없음 · 조회 기간 최대 90일 · 미래 조회 불가 · 페이지 100행 · CSV 단일 형식 · 상세는 링크로 공유 불가 |

| 상태 계열 | 2xx · 4xx · 5xx 셋. 3xx 는 2xx 로 접힌다 |

| 파일 용량 제한 · 업로드 확장자 | 해당 없음 — 파일 입력이 없다 |
