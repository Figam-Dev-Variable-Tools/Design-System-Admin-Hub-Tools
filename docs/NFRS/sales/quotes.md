## 7. 유지보수성

| 항목 | 요구 | 이 자원의 값 |
|---|---|---|
| 코드 확장성 | 공통 준수 | 상태 전이·계약 생성·청구 생성의 판정이 전부 **순수 술어 함수**다(`quoteStatusChangeBlock` · `contractDraftBlock` · `billingCreateBlock`). 버튼의 존재 조건과 저장의 거절 조건이 **같은 함수**를 읽어 '눌리는데 실패하는 버튼'이 생길 수 없다 |
| 거절이 문자열인 이유 | 화면이 거절 사유를 다시 지어내지 않는다 | 술어가 boolean 이 아니라 **사유 문자열**을 돌려준다 — 화면마다 같은 거절을 다르게 설명하지 않게 한다 — `apps/admin/src/pages/sales/quotes/types.ts:274-283` |
| 화면 확장성 | 열·필터·상태값이 늘 때 견디는가 | 상태는 데이터(`QUOTE_STATUS_OPTIONS`)로 들고 있고 전이표도 `Record` 다 — 칸을 하나 늘려도 화면 코드를 고치지 않는다. 조회 조건은 주소 쿼리스트링이 소유한다 |
| 공통 컴포넌트 사용 여부 | 페이지가 UI 를 손수 조립하지 않는다 | 목록은 공통 껍데기(`CrudListShell`), 폼은 공통 컨트롤러(`useCrudForm`)를 쓴다. **상세는 껍데기가 없어 카드·배너·버튼 줄을 직접 배치한다** — 상태 관리 카드와 경고 배너 3종이 이 화면 고유 조립이다 |
| 어휘를 개명하지 않은 근거 | 같은 사실에 두 이름을 만들지 않는다 | `ordered` 를 '계약'으로 바꾸지 않았다 — 이 상태는 **청구의 근거이기도 해서**('수주로 전환된 견적만 청구할 수 있어요.') 개명하면 그 문장이 거짓이 된다. 라벨은 한 곳(`QUOTE_ORDERED_LABEL`)에서만 정한다 — `apps/admin/src/pages/sales/quotes/types.ts:177-188` |

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 외부 시스템 연계 | 없음. 견적서는 앱 안에서 인쇄된다(전자서명·메일 발송 연동이 없다) |
| API 제약 | **백엔드가 없다.** 서버가 맡을 계약: `GET/POST /api/sales/quotes` · `GET/PUT/DELETE /api/sales/quotes/:id` · `POST /api/sales/quotes/:id/order`(계약과 한 트랜잭션) · `POST /api/sales/quotes/issue`(문의 → 견적, **이미 발행된 문의가 섞이면 409**) — `apps/admin/src/pages/sales/quotes/data-source.ts:175-176,201-202` |
| 등록 경로 | **없다.** `/sales/quotes/new` 는 차단 안내 화면이며 어떤 권한으로도 폼이 열리지 않는다. 견적은 문의에서만 발행된다 |
| 되돌리는 전이 | **없다.** `accepted → sent` 를 열면 그 견적으로 이미 만든 계약·청구가 무엇이 되는지 답할 수 없고, 되돌릴 트랜잭션도 없다 — `apps/admin/src/pages/sales/quotes/types.ts:226-230` |
| 품목 수 상한 | 30행 — `apps/admin/src/pages/sales/quotes/types.ts:131` |
| 문자 수 상한 | 담당자 40자 · 비고 500자 · 품목명 60자(`QUOTE_ITEM_NAME_MAX`) |
| 세액 계산 규칙 | **라인별 반올림 후 합산**으로 고정한다(문서 규칙). 면세·영세율은 세액 0 — `apps/admin/src/pages/sales/quotes/types.ts:352-366` |
| 기본 유효기간 | 발행일 +30일(국내 관례) — `apps/admin/src/pages/sales/quotes/types.ts:421` |
| 파일 업로드 | 해당 없음 — 이 자원에 파일 입력이 없다 |
| 거래처 연결 | `accountId` 는 빈 값을 허용한다(문의 발행 경로를 막지 않기 위해). 그 대가로 **미등록 견적은 거래처 상세의 견적 이력에서 빠진다** — 폼이 그 사실을 warning 으로 드러낸다 |
