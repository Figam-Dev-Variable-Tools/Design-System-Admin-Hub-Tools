# 연혁 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 연혁 |
| 화면 ID | `SCR-COMPANY-HISTORY` · `SCR-COMPANY-HISTORY-FORM` |
| 메뉴 경로 | 기업 관리 > 연혁 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [목록](../../FSD/company/history/index.md) · [등록·수정](../../FSD/company/history/form.md) |

**이 자원의 성격** — **기업 관리 목록형 자원의 정본이다.** [ESG](esg.md) · [채용 공고](careers.md)가 같은 공용 목록·폼 껍데기를 쓰므로 그 문서들은 이 문서와의 **차이만** 적는다. [인증서/특허](certificates.md)는 드래그 재정렬 때문에 껍데기를 쓰지 못하고 같은 모양을 손으로 조립한다.
이 계열의 결정 셋 — ① **쓰기가 실제로 반영되고 없는 대상은 충돌로 막힌다** ② **쓰기 권한이 없으면 컨트롤이 사라지고 폼은 열리지 않는다** ③ **저장 충돌·422·멱등키 방어가 있다**. 콘텐츠 관리 여섯 자원과 정반대다.
