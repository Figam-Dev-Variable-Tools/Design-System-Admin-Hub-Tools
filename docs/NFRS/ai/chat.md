# 새 채팅 — 비기능 명세서

| 항목 | 내용 |
|---|---|
| 자원명 | 새 채팅 |
| 화면 ID | `SCR-AI-CHAT` |
| 메뉴 경로 | AI > AI 에이전트 > 새 채팅 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |
| 관련 기능 명세서 | [새 채팅](../../FSD/ai/chat/index.md) |

**이 자원의 성격** — 대화형 조회 화면이다. 조회 대상 데이터는 **아무것도 바꾸지 않지만**, 대화·메시지라는 자기 상태를 만든다. 그 상태가 **브라우저 메모리에만 있다**는 것이 이 자원의 비기능 요구를 지배한다. 나머지 축은 ① 없는 기능을 있는 척하지 않기 ② 조회 결과의 정직함 ③ 한국어 입력 안전 셋이다.

**AI 에이전트 2화면은 같은 조회·같은 저장소를 공유한다** — 대화 목록의 값은 [대화 목록](conversations.md) 이 갖고, 겹치는 부분은 이 문서를 가리킨다.

아래 각 절은 [공통 기준](../_common.md) 대비 **이 자원만의 차이**를 적는다.

## 8. 제약사항

| 항목 | 내용 |
|---|---|
| 외부 시스템 연계 | **언어 모델이 연결되어 있지 않다.** 지금 답하는 것은 결정적 파서이며, 요약·분석·예측·추천은 거절한다 |
