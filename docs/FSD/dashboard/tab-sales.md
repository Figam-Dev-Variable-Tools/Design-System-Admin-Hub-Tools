# 대시보드 — 영업 탭

## 1. 화면 개요

| 항목 | 내용 |
|---|---|
| 화면명 | 대시보드 영업 탭 |
| 화면 ID | `TAB-DASHBOARD-SALES` |
| 메뉴 경로 | 일반 관리 > 대시보드 > (업무 영역) 영업 |
| 작성자 | Admin Hub 문서화 |
| 작성일 | 2026-07-22 |
| 버전 | 1.0 |

**화면 목적** — 영업 담당자가 새로 들어온 영업 문의와 계약 진행 상황을 한눈에 보고, 대기 중인 건으로 바로 들어간다.

**업무 배경** — 영업은 문의 → 견적 → 계약 → 청구 → 프로젝트가 하나의 흐름이다. 그 흐름의 **앞머리 둘**(문의·계약)이 막히면 뒤가 전부 밀린다. 이 탭은 그 두 지점의 대기 건수를 먼저 보여 준다.

**화면 설명** — 대시보드의 업무 탭 중 하나다. 오늘의 할일 카드(4항목)와 리스트 카드 2장(`최근 문의` · `계약 대기`)으로 이루어진다. 탭 줄과 통계는 부모 화면([대시보드](index.md))의 것이다.

**주요 사용자** — `dashboard.tab.sales` 권한을 가진 역할(영업 담당자).

**사용 시나리오**

1. 영업 담당자가 대시보드에서 영업 탭을 고른다 → 문의대기 건수를 눌러 영업 문의 목록으로 간다.
2. `계약 대기` 카드에서 어떤 건이 서명을 기다리는지 훑고 계약 목록으로 넘어간다.

**선행 조건** — 로그인 · `dashboard.tab.sales` 권한. 할일·리스트 카드는 각각 `dashboard.todo` · `dashboard.lists` 권한이 더 필요하다.

**후행 처리** — 없다. 읽고 이동할 뿐이다.

**관련 화면** — 부모 [대시보드](index.md) · 다른 탭 [상품](tab-products.md) · [문의](tab-inquiries.md) · 이동 대상 `SCR-SALES-INQUIRIES` · `SCR-SALES-CONTRACTS`.

**관련 메뉴** — 사이드바 항목이 아니다. 대시보드 안의 탭이다.

**관련 기능** — 권한 관리 화면의 위젯 토글 `탭 — 영업`. 영업 관리 메뉴 아래 여섯 화면이 이 탭이 말하는 숫자의 원천이다.

**관련 API** — 영업 탭의 할일·리스트 조회 1건(활성 탭을 키로 삼는다). **백엔드가 없다** — 지금 보이는 건수와 행은 고정 픽스처다 — `apps/admin/src/pages/dashboard/api.ts:163-197`.

**참고 문서** — [ID 규약](../id-convention.md) · [인벤토리](../inventory.md)

**구현 파일** — `apps/admin/src/pages/dashboard/components/DashboardTabPanel.tsx` (탭 정의 `apps/admin/src/pages/dashboard/types.ts:21` · 권한 매핑 `useDashboardTabs.ts:20` · 데이터 `api.ts`)

## 4. 기능 명세

**플랜 축이 이 탭에 닿을 수 있다** — 영업 파이프라인은 상위 플랜부터 열리는 모듈이다(`apps/admin/src/shared/entitlements/plan.ts:213-218`). 그 모듈이 잠긴 계정에서 이 탭의 링크를 누르면 도착 화면이 잠금 화면으로 대체된다. **탭 자체가 사라지지는 않는다** — 대시보드 위젯은 권한 축에만 걸려 있고 플랜 축에는 걸려 있지 않다(코드에서 확인한 사실이다).

## 8. 화면 상태(State)

| 상태 | 화면 표시 | 버튼 활성화 여부 | Loading 여부 | 사용자 안내 메시지 |

| 최초 로딩 | 할일 카드는 골격 + 스켈레톤, 리스트는 자리표시 2장 | 클릭 × | Y | 없음 |
| 데이터 있음 | 할일 4항목 + 리스트 2장 | 클릭 ○ | N | 없음 |

| 조회 실패 | 패널이 배너로 대체된다 | — | N | `대시보드 데이터를 불러오지 못했어요. 잠시 후 다시 시도해 주세요.` |
| 탭 전환 중 | 이전 탭 내용을 남기지 않고 스켈레톤 | 클릭 × | Y | 없음 |
| 저장 중 | 해당 없음 — 저장 기능이 없다 | — | N | — |
| 읽기 전용(권한) | 이 탭은 언제나 읽기 전용이다 | — | N | — |

**0 은 '모름'이 아니다** — 이 탭의 건수는 조회가 성공해 돌려준 값이다. 값을 모를 때 `—` 를 그리는 위젯은 대시보드의 문의 현황뿐이다([대시보드](index.md) §4.6).

**권한이 없을 때 화면이 어떻게 되는가** — **탭이 사라진다.** 비활성이 아니라 목록에서 빠진다 — `apps/admin/src/pages/dashboard/useDashboardTabs.ts:7-8`. 보고 있던 중에 권한이 꺼지면 남아 있는 첫 탭으로 흘러간다. 할일·리스트 카드는 각자의 위젯 권한으로 따로 사라진다.
