## 7. 예외 처리

동작 칸은 둘뿐이다: `조회 결과 없음` · `권한 없음`. 나머지 열 상황은 명세가 스스로 `해당 없음` 을
선언했으므로 테스트를 요구하지 않는다 — **그 부재가 정답이다.**
`서버 오류` 행의 `〃` 는 윗행(`API 오류` = `해당 없음`)을 물려받는다.

| 발생 상황 | 사용자 메시지 | 시스템 처리 | 복구 방법 |
|---|---|---|---|
| 조회 결과 없음 | `표시할 항목이 없습니다` 안내를 렌더한다 | 표 골격은 남긴다 | 조건을 바꾼다 |

| 삭제 실패 | 해당 없음 — 삭제 기능이 없다 | — | — |
| API 오류 | 해당 없음 — 실패 표시는 상위 배너가 담당한다 | — | — |
| 서버 오류 | 〃 | — | — |
| 네트워크 오류 | 해당 없음 — 상위 배너가 담당한다 | — | — |
